Extra System.Array Methods Example: The slides list several array methods, but they do not give one complete demo that uses them together. Use the following example as a reference before solving the exercises.

Example Program

class Program { static void Main(string[] args) { int[] numbers = { 8, 3, 6, 1 }; System.Console.WriteLine("Length = " + numbers.Length); System.Console.WriteLine("Value at index 2 = " + numbers.GetValue(2)); numbers.SetValue(10, 1); Array.Sort(numbers); Array.Reverse(numbers); int[] copied = new int[6]; numbers.CopyTo(copied, 1); foreach (int item in copied) System.Console.Write(item + " "); } }

What To Notice

- Length returns number of elements - GetValue reads a value by index - SetValue changes a value by index - Array.Sort sorts from small to large - Array.Reverse flips the current order - CopyTo copies into another array starting at a chosen index
