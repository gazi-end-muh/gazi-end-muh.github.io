Example Program

using System; using System.IO; class Program { static void Main() { string path = "notes.txt"; string[] notes = { "meet team", "review array task", "finish loop practice" }; try { File.WriteAllLines(path, notes); string[] savedNotes = File.ReadAllLines(path); for (int i = 0; i < savedNotes.Length; i++) { Console.WriteLine((i + 1) + ". " + savedNotes[i]); } } catch (IOException ex) { Console.WriteLine("File error: " + ex.Message); } } }

Why It Matters

- array data is written as lines - a loop reads lines back one by one - a method from string processing is applied - try/catch prevents crashes - this is close to reading and transforming text for a Caesar cipher style program
