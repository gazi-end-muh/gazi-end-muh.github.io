Example Program

class Program { static int Sum(int a, int b) { return a + b; } static void PrintLine(string label, int value) { System.Console.WriteLine(label + value); } static void Main(string[] args) { int result = Sum(4, 7); PrintLine("Result = ", result); } }

What To Notice

- Each method belongs to the class - Sum returns an int value - PrintLine is void because it only prints - Parameter count, order, and types must match - Main can call static methods directly in the same class
