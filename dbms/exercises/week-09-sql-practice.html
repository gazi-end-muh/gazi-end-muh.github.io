Cursor Example Query

Follow the same style used on the programming pages: first inspect a nearby example, then solve the exercise. The query below is not the answer to every question, but it shows the expected SQL format and alias style.

SELECT first_name || ' ' || last_name AS employee_name, salary + bonus AS total_compensation FROM employees ORDER BY total_compensation DESC;
Expected idea: - Build a readable full name with || - Use an expression in the SELECT list - Assign aliases with AS - Sort final output with ORDER BY
